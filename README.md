# FortiGate Firewall Automation with Ansible

This project demonstrates how to automate FortiGate firewall policies using Ansible and Infrastructure-as-Code principles.

It is designed as a **lab / showcase project** for DevOps, Network Automation, and System Engineering roles.

---

## Features

- Automated deployment of firewall policies
- Structured policy groups (F2, F25, F3, F4)
- Loop-based policy management (scalable & clean)
- Environment-based authentication (no secrets in repo)
- Policy ordering (critical for firewall logic)

---

## Tech Stack

- Ansible
- FortiOS (FortiGate)
- YAML
- Infrastructure as Code (IaC)
